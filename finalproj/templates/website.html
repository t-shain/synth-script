Synth Script

Text-To-MIDI Converter






HOW DOES IT WORK?

  1. A script is typed by user and then sent to the local Raspberry Pi directory.
  2. The string is changed into a MIDI file using the Mido library
  3. The MIDI file is turned into an audio file using Fluidsynth
  4. The audio file is sent back to the computer.
  5. The audio file is sent back to the computer.

Synth Scripts algorithm uses the patterns in the English language to develop melodies. Vowels, string length, non vowels, and punctuation are the four main features our algorithm uses to produce melodies. Vowels are placed where chords will be played. String length determines the tempo of the melody. Non vowel characters are used for 1-3 note patterns. And if a string ends with '!' or '...' it will change the velocity of all notes (velocity refers to how hard each not is hit/played).


Plans for the Future

This is just a functioning prototype for a string to MIDI conversion system. Plans for further development include:

  • More instrument to choose from
  • Add effects like reverb, chorus, or delay
  • Added option to make songs for drums
  • More complexity to website and interactivity
  • Database investment to handle more traffic
